## How to Add Your Profile Picture

To add your actual profile picture:

1. Place your image file in the `src/assets/images/` folder (create the folder if it doesn't exist)
2. Update the about component HTML:
   ```html
   <!-- Replace the emoji div with this image tag -->
   <img src="assets/images/your-profile-picture.jpg" alt="Your Name" class="w-full h-full object-cover">
   ```
